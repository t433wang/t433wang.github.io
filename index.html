CS488 Project: Ray Tracer

A ray traver written in C++ that render scenes with a Lua file. Written for CS488 Computer Graphics in University of Waterloo

Objectives

Objectives for this project include

  • Anti-Alias
  • Soft Shadow
  • Refraction
  • Glossy Reflection
  • Phong Shading
  • Texture Mapping
  • CSG
  • Multi-Threading

    • Anti-Alias

    Emit 20 rays for each pixel, each ray is in the range of a pixel

    Use Stochastic sampling to randomize the direction

    Return the average colour of the 20 rays

    No Anti-Alias

    20x Anti-Alias

    Soft Shadowing

    Randomly sample the points on the area light source

    Render hard shadows

    Sum up and averaging the shadow contributions

    No Soft Shadow

    10x Soft Shadow

    30x Soft Shadow

    100x Soft Shadow

    Refraction

    Use Snell's Law to calculate the direction of refracted ray

    Recursively trace the refracted ray

    Blend the color from refraction and reflection by Fresnel's Law

    Refract coefficient: 1.1

    Refract coefficient: 1.0

    Glossy Reflection

    Randomly create r', which is the perturbation from reflection ray r, using a uniform distribution

    Average colors of the sampled r'

    Glossy Reflection

    Mirror Reflection

    Phong Shader

    Calculate vertex normal of each vertex v by averaging the normals of faces that use v

    Calculate the barycentric coordinates of the intersection point

    Interpolate normals across the polygon using the barycentric coordinates (Gouraud techniques)

    Flat Shading

    Phong shading

    Texture Mapping

    Get ray intersection (x,y,z)

    For each primitive, map (x,y,z) to (u,v)

    Return (u,v) pixel information from image and use it for rendering
