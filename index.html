CS488 Project: Ray Tracer

A ray traver written in C++ that render scenes with a Lua file. Written for CS488 Computer Graphics in University of Waterloo

Objectives

Objectives for this project include Anti-Aliasing, soft shadowing, refraction, glossy reflection, Phong shading, texture mapping, CSG, and Multi-Threading.

Anti-Aliasing

Emit 20 rays for each pixel, each ray is in the range of a pixel. Use Stochastic sampling to randomize the direction. Return the average colour of the 20 rays.

No anti-aliasing
